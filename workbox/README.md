# workbox

My workbox guide

## Requirements

- [Workbox](https://developers.google.com/web/tools/workbox/)

## Installation

```bash
# Install workbox-cli globally
$ npm install -g workbox-cli

# Generate workbox-config.js
$ workbox wizard
```
